### Sobre o Projeto

Neste projeto, você utilizei o Python para explorar dados relacionados aos sistemas de compartilhamento de bicicletas nas três maiores cidades dos Estados Unidos: Chicago,
 Nova Iorque e Washington. Escrevendo um código para importar dados e responder a perguntas interessantes sobre eles, 
por meio da computação de estatísticas descritivas. Tambem escrevi um script que recebe uma entrada bruta para criar uma experiência interativa no terminal, 
a fim de apresentar estas estatísticas.
